Basic movement algorithm

ROW 1

A basic movement algorithm for an NPC will update the character's position to move towards a target position. The input to the algorithm will include the NPC's current position, the target position, and the speed of the NPC. The output will be the updated position of the NPC based on the desired velocity.

// 
// Define the NPC's initial position
current_position: Vector2 or Vector 3

// Define the target position 
target_position: Vector2 or Vector 3

// Define the speed of the NPC 
speed = float

// Update function called every frame 
function update(): 
// Calculate the direction vector towards the target 
direction= target_position - current_position

// Normalize the direction vector
// if Vector2 is:
magnitude = √(x2 + y2)
direction.x = direction.x / magnitude 
direction.y = direction.y / magnitude 

// if Vector3 is:
magnitude = √(x2 + y2 + z2)
direction.x = direction.x / magnitude 
direction.y = direction.y / magnitude
direction.z = direction.z / magnitude

// Calculate the velocity vector 
velocity = direction * speed 

// Update the NPC's position based on velocity 
current_position += velocity
