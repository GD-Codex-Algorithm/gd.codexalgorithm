Kinematic wandering moevement

The kinematic wander movement allows a character to move in a direction based on its current orientation, always at maximum speed. The steering behaviour modifies the character's orientation randomly, resulting in a meandering movement as it moves forward. This behaviour simulates a wandering, explorative motion that gives the character a more natural and unpredictable path. The character moves only forward in each frame, maintaining the direction it was facing in the previous frame, while its orientation changes gradually to create a wandering effect.

Pseudo Code

        

class KinematicWander:
    character: Static
    maxSpeed: float
    maxRotation: float

    function getSteering() -> KinematicSteeringOutput:
        result = new KinematicSteeringOutput()

        # Generate a random rotation within the allowed maximum rotation
        wanderOrientation = randomBinomial() * maxRotation

        # Update the character's orientation by adding the wander orientation
        character.orientation += wanderOrientation

        # Calculate the direction vector based on the updated orientation
        direction = Vector(cos(character.orientation), sin(character.orientation))

        # Calculate the velocity vector based on the direction and maximum speed
        result.velocity = direction * maxSpeed

        # Update the character's position based on the velocity
        character.position += result.velocity

        # Set the character's new position and orientation
        set_character_position(character.position)
        set_character_orientation(character.orientation)

        # No rotational movement needed in the steering output
        result.rotation = 0
        return result

    # Helper function to generate a random number between -1 and 1
    function randomBinomial() -> float:
        return random() - random()
